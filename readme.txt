== Description ==
This Plugin will censor your site on January 18th between 8am and 8pm local time  in protest of SOPA and PIPA. It has no admin settings and is super easy to use.

== Installation ==

= Automattic Install (recommended) =
1. Login to your Dashboard as an admin.
2. Once logged in select Plugins > Add New from the main menu.
3. Next search for "sopa-blackout" (no quotes) and click the install button.
4. Update the MobileESP for WordPress settings with the url you want mobile viewers to be redirected to.

= Manual Install =
1. Extract the contents of sopa-blackout.zip and upload to the wp-content/plugins folder.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Update the MobileESP for WordPress settings with the url you want mobile viewers to be redirected to.
